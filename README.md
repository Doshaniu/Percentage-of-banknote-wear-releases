# Анализ ветхости банкнот

Программа анализирует логи устройств самообслуживания (УС) и рассчитывает процент ветхих банкнот.

## Требования

- Windows 10/11
- Больше ничего не нужно — всё включено в exe файл

## Установка

1. Скачать `BanknoteAnalyzer.exe` и `install.bat` в одну папку
2. Запустить `install.bat` один раз

## Использование

1. Запустить `BanknoteAnalyzer.exe`
2. В поле **«Папка с данными»** выбрать папку с выгруженными архивами логов (`.7z`)
3. В поле **«Папка для отчёта»** выбрать куда сохранить результат
4. Нажать **«Запустить анализ»**
5. После завершения нажать **«Открыть папку с отчётом»**

> ⚠️ При первом запуске анализ может занять 3-5 минут — это нормально.

## Результат

Excel файл `results_ДАТА_ВРЕМЯ.xlsx` со столбцами:

| УС | Дата | Процент |
|---|---|---|
| 1113 | 01.01.26 | 3.3% |

Чем ниже процент — тем лучше состояние банкнот в устройстве.
